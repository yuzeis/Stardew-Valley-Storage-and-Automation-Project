# SVSAP / SVSAPME Ver1.4.1-rc1.0 发布候选版

内部版本：`1.4.1-rc1.0`

这是 1.4.1 版本线的首个发布候选版，用于正式版前的广泛实机测试。更新前请备份重要存档，并确保所有联机玩家使用完全相同的 SVSAP 与 SVSAPME 版本。

## 本次更新

- 统一重构全部自定义 GUI，加入紧凑的红黄绿状态灯，修正槽位与物品图标错位，恢复原版品质星，并仅对大数量使用 K/M 缩写。
- 改进网络终端、存储驱动器、3x3 传输过滤、导入导出方向、合成确认与合成监视流程。
- 存储接口与箱子读取逻辑只处理预期的相邻容器，不再扫描地图上的无关箱子。
- 导入器与导出器可按方向和过滤规则与原版箱子交互。
- 多人远程操作加入持久化对账、幂等请求、主机权威箱锁与重连恢复。
- 单方块农场、酒桶与陈酿桶提供真实输入、输出和升级槽，独立工作格进度与 ETA，网络自动输入输出，以及能源和经济统计。
- 单方块农场支持混种、地块锁定、单格铲除与清空全部确认；铲除和清空不会返还投入物或进度。
- 修复咖啡批量输入、品质卡自动拉取、畸形缓冲恢复与跨夜处理进度结算。

## 安装要求

- 需要 Stardew Valley 1.6.15 或更高版本，以及 SMAPI 4.5.2 或更高版本。
- SVSAPME 需要 SVSAP `1.4.1-rc1.0` 或更高版本。
- 安装前删除旧的 `SVSAP` 与 `SVSAPME` 文件夹，再放入新文件夹；不要直接合并旧版目录。
- 多人联机必须让主机和全部 farmhand 同步更新。

本版本并非 1.4.1 最终正式版。请先备份重要存档；反馈问题时请提供 SMAPI 日志与完整复现步骤。
