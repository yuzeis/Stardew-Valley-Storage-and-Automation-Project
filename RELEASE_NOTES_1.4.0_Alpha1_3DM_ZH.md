# SVSAP / SVSAPME Ver1.4.0Alpha1

内部版本：`1.4.0-alpha.1`

这是 1.4.0 线的 Alpha 测试版本。新增机器内容归属在 SVSAPME。

## 更新重点

- 新增 SVSAPME 独立机器：铜/钢/金/铱单方块酒桶。
- 新增 SVSAPME 独立机器：铜/钢/金/铱单方块陈酿桶。
- 四级容量为 16 / 64 / 144 / 256 个内部槽位。
- 新增处理机网格 GUI，每格显示产物、进度、ETA，并支持收取。
- 新增处理机输入缓冲，并为存储驱动器、导入/导出器、农场、酒桶和陈酿桶 GUI 补充紧凑布局边界检查。
- 处理机网络自动输入默认关闭；schema v3 迁移只关闭旧档中“全量可拉取且无过滤”的危险默认值，保留玩家主动配置的过滤规则。
- 单方块酒桶使用原版风味酒/果汁创建路径；酒桶 ETA 采用明确的每日 1200 游戏分钟推进口径。
- 农场手空手打开存储驱动器、导入器、导出器时会打开主机快照远程配置菜单，不再进入本地空菜单。
- 农场手处理机交付和会消耗手持物的机器动作加入持久化 ACK/escrow 路径，降低断线导致的丢失或复制窗口。
- 修复未完成处理槽位回收问题：拆除、异常回收或降级溢出未完成槽位时返还投入物，不会提前生成未来产物。
- SVSAP 与 SVSAPME 版本字段统一为 `1.4.0-alpha.1`。

## 安装

1. 安装 SMAPI 4.5.2 或更新版本。
2. 安装 `SVSAP 1.4.0-alpha.1.zip`。
3. 如需机器与能源扩展，安装 `SVSAPME 1.4.0-alpha.1.zip`。
4. 多人联机时所有玩家应使用相同版本的 SVSAP 与 SVSAPME。

Alpha 测试版建议先备份存档。
