# SVSAP / SVSAPME 1.4.0-alpha.1.2 修复更新

本次版本用于 1.4 Alpha 的下一轮实机测试，重点修复多人联机物品核对、终端状态串扰、部分界面性能和单方块农场锁定地块补种问题。

## SVSAP 修复

- 远程终端存入物品、存储元件、样板和升级卡等远程操作物品会持续安全保留，直至主机确认结果。
- 断线后会使用原事务继续核对，避免操作物品直接丢失或被无条件返还。
- 其他玩家使用终端时，不会再改变当前玩家的物品页、合成数量或品质策略。
- 优化存储驱动器与样板供应器界面，减少打开菜单后的重复计算。

## SVSAPME 修复

- 农场工人的机器投料在断线重连后会与主机核对，避免重复返还。
- 主机处理机器操作异常时会向客户端返回明确结果，并加入有限次数的超时重试。
- 单方块农场会优先满足锁定地块的指定种子，再为普通地块自动补种。
- 新增混种、锁定地块、再生作物和收获输出的生产路径自测。

## 安装说明

1. 需要《星露谷物语》1.6.15 或更高版本。
2. 需要 SMAPI 4.5.2 或更高版本。
3. 请先安装 SVSAP 1.4.0-alpha.1.2，再安装 SVSAPME 1.4.0-alpha.1.2。
4. 联机时，主机和所有玩家必须使用相同版本。

本版本仍属于 Alpha 测试版本，建议先备份重要存档。
